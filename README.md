## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

### Build for Production

```bash
npm run build
npm start
```

## Customization

- Update personal information in components
- Add your GitHub and LinkedIn URLs in `Hero.tsx` and `Contact.tsx`
- Modify color scheme in `tailwind.config.ts`
- Add more projects in `Projects.tsx`
- Update experience timeline in `Experience.tsx`

## Tech Stack

- Next.js 15
- React 19
- TypeScript
